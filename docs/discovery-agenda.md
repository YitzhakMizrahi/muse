# Discovery Agenda

The project is now in documentation-first discovery mode.

Before more implementation work, we should settle the following:

1. exact v1 artifact
2. session input model
3. output structure
4. recommendation logic
5. user journey from start to finished artifact
6. visual language and content tone
7. handoff target to downstream builders or agents
8. what is explicitly out of scope

## Working Method

For each topic:

1. discuss options
2. choose a direction
3. record the decision in docs
4. move to the next topic

## Rule

Do not add more product code until the core product definition is stable enough to defend.
